🏸 Badminton Itapetininga

Projeto desenvolvido para apoiar e incentivar o trabalho voluntário de uma família dedicada ao desenvolvimento do badminton em Itapetininga/SP. O site tem como objetivo divulgar informações sobre o projeto, incentivar novos participantes e fortalecer a comunidade esportiva local.

📌 Sobre o Projeto

O badminton é um esporte que promove disciplina, inclusão social, saúde e trabalho em equipe. Este site foi criado para facilitar a divulgação do projeto esportivo, apresentar informações sobre treinos e aproximar a comunidade das ações voluntárias realizadas em Itapetininga.
